## 1. Installer l'application sur votre telephone

FleetMaster Pro est une application web progressive (PWA). Pas besoin de passer par l'App Store ou le Play Store : vous l'installez directement depuis votre navigateur.

### Sur Android (Chrome)

1. Ouvrez **Chrome** sur votre telephone.
2. Allez sur l'adresse FleetMaster que votre entreprise vous a communiquee (exemple : `https://app.fleetmaster.fr`).
3. Connectez-vous avec vos identifiants chauffeur.
4. Une banniere bleue apparait en haut de l'ecran : **"Installer FleetMaster"**.
5. Appuyez sur **Installer**.
6. L'icone FleetMaster apparait sur votre ecran d'accueil. Vous pouvez l'ouvrir comme n'importe quelle application.

> Si la banniere n'apparait pas :
> - Appuyez sur les **trois points** en haut a droite de Chrome.
> - Selectionnez **"Installer l'application"** ou **"Ajouter a l'ecran d'accueil"**.

### Sur iPhone (Safari)

1. Ouvrez **Safari** (important : ca ne fonctionne pas avec Chrome sur iPhone).
2. Allez sur l'adresse FleetMaster Pro.
3. Connectez-vous avec vos identifiants.
4. Appuyez sur le bouton **Partager** (le carre avec la fleche vers le haut, en bas de l'ecran).
5. Faites defiler la liste et appuyez sur **"Sur l'ecran d'accueil"**.
6. Appuyez sur **Ajouter**.
7. L'icone FleetMaster apparait sur votre ecran d'accueil.

### Verifier que l'installation a fonctionne

- Ouvrez l'application depuis l'ecran d'accueil.
- Elle doit s'ouvrir **en plein ecran** (sans barre d'adresse du navigateur).
- Vous recevrez des notifications push pour les alertes importantes (maintenance, documents a renouveler, etc.).

## 2. Scanner le QR code de votre vehicule

Chaque vehicule de la flotte dispose d'un **QR code unique** (generalement colle sur le tableau de bord ou le pare-brise).

### Comment scanner

1. Ouvrez l'**appareil photo** de votre telephone (pas besoin d'application speciale).
2. Pointez l'objectif sur le QR code du vehicule.
3. Un lien apparait : appuyez dessus.
4. Vous arrivez sur la **page du vehicule** qui affiche :
   - La marque, le modele et l'immatriculation du vehicule.
   - Trois choix sous forme de cartes :

| Carte | Description | Acces |
|-------|-------------|-------|
| **Inspection pre-depart** | Controler l'etat du vehicule avant de rouler | Tous les chauffeurs |
| **Saisir un plein** | Enregistrer un ravitaillement en carburant | Tous les chauffeurs |
| **Carnet d'entretien** | Historique complet du vehicule | Gestionnaires uniquement (connexion requise) |

5. Appuyez sur la carte souhaitee.

> **Bon a savoir** : Le scan QR fonctionne meme si vous n'etes pas connecte a l'application. Les formulaires d'inspection et de carburant sont accessibles sans compte.

## 3. Faire l'inspection pre-depart

L'inspection pre-depart est obligatoire. Elle permet de verifier l'etat du vehicule avant chaque prise de service. Deux methodes sont disponibles.

### Methode 1 : Via le QR code (recommandee)

1. Scannez le QR code du vehicule (voir section 2).
2. Appuyez sur la carte **"Inspection pre-depart"**.

### Methode 2 : Via l'application chauffeur

1. Ouvrez FleetMaster depuis votre ecran d'accueil.
2. Sur la page d'accueil, appuyez sur le bouton vert **"Checklist depart"**.

### Deroulement de l'inspection

L'inspection se fait en **3 etapes** :

#### Etape 1 — Informations generales

- **Kilometrage** : saisissez le kilometrage affiche au compteur. Il doit etre superieur ou egal au dernier kilometrage enregistre.
- **Lieu** : indiquez ou vous etes (par defaut : "Depot").
- **Niveau gasoil** : ajustez le curseur pour indiquer le niveau de carburant.
- Si votre vehicule est un **poids lourd** : un champ AdBlue apparait.
- Si votre vehicule est un **poids lourd frigo** : des champs GNR et temperatures des compartiments apparaissent.
- Appuyez sur **Continuer**.

#### Etape 2 — Points de controle

Chaque element est a evaluer. Pour chacun, appuyez sur l'un des trois boutons :

| Bouton | Signification |
|--------|---------------|
| **OK** | Tout est normal |
| **Attention** (triangle jaune) | Quelque chose a surveiller, mais on peut rouler |
| **Critique** (rond rouge) | Probleme grave, a signaler immediatement |

Les points sont organises par priorite :

- **Points critiques** : freinage, pneus, eclairage, direction (et moteur/temperature frigo pour les PL frigo).
- **Points importants** : niveaux, essuie-glaces, avertisseur, ceintures, documents, tachygraphe (PL).
- **Equipement** : kit de securite (triangle, gilet).
- **Etat de proprete** : carrosserie, vitres, habitacle (Propre / Sale / Abime).

Appuyez sur **Continuer**.

#### Etape 3 — Validation

- Un **recap** visuel affiche le score global et les eventuels problemes detectes.
- Saisissez votre **nom et prenom**.
- Appuyez sur **Valider**.
- Un **numero de ticket** s'affiche : notez-le, c'est votre preuve d'inspection.

> **Score** : 100% = Excellent. Un point critique non-OK fait chuter le score de 25 points. Un avertissement retire 10 points. Votre gestionnaire de flotte est automatiquement notifie si le score est bas.

## 4. Saisir un plein de carburant

Deux methodes pour enregistrer un plein.

### Methode 1 : Via le QR code (recommandee sur le terrain)

1. Scannez le QR code du vehicule.
2. Appuyez sur **"Saisir un plein"**.

### Methode 2 : Via l'application chauffeur

1. Ouvrez FleetMaster depuis votre ecran d'accueil.
2. Appuyez sur le bouton vert **"Saisir un plein"** ou allez dans l'onglet **Carburant** en bas de l'ecran.

### Remplir le formulaire (via QR code)

Le formulaire QR code est le plus complet. Il permet d'enregistrer **jusqu'a 3 carburants en une seule fois** (par exemple : Gasoil + AdBlue + GNR).

Pour chaque carburant :

1. **Type de carburant** : selectionnez dans la liste (Gasoil, AdBlue, GNR, Essence, etc.).
2. **Litres** : saisissez la quantite.
3. **Kilometrage** : saisissez le kilometrage au compteur. Le GNR (groupe frigo) n'a pas besoin de kilometrage.
4. **Prix** : optionnel. Si vous etes a un automate (type AS24), vous n'avez pas forcement le prix.

Pour ajouter un deuxieme carburant, appuyez sur **"Ajouter un carburant"** (maximum 3 par session).

Renseignez ensuite :

5. **Nom de la station** : optionnel (Total, Shell, AS24...).
6. **Votre nom et prenom** : obligatoire.

Verifiez le **recapitulatif** en bas de page, puis appuyez sur **"Enregistrer tout"**.

Un **numero de ticket** s'affiche pour confirmer l'enregistrement.

### Remplir le formulaire (via l'application)

Le formulaire de l'application est simplifie :

1. **Kilometrage actuel** : obligatoire.
2. **Type de carburant** : Diesel, Essence ou AdBlue.
3. **Litres** : obligatoire.
4. **Prix au litre** : obligatoire.
5. **Station-service** : optionnel.

Le cout total se calcule automatiquement. Appuyez sur **"Enregistrer le plein"**.

## 5. Signaler un incident

En cas d'accident, de panne, d'accrochage ou de tout autre probleme, signalez-le immediatement.

### Acceder au formulaire

1. Ouvrez FleetMaster depuis votre ecran d'accueil.
2. Sur la page d'accueil, appuyez sur le bouton rouge **"Signaler incident"**.
3. Ou bien, dans la barre de navigation en bas, appuyez sur l'onglet **Incident**.

### Remplir le signalement

#### 1. Vehicule concerne

Votre vehicule assigne s'affiche automatiquement. Appuyez dessus pour le selectionner.

#### 2. Type d'incident

Choisissez parmi les options proposees :

| Type | Quand l'utiliser |
|------|------------------|
| **Accident** | Collision avec un autre vehicule ou un obstacle |
| **Panne** | Probleme mecanique (moteur, boite de vitesses...) |
| **Accrochage** | Dommage materiel mineur |
| **Incendie** | Depart de feu |
| **Autre** | Tout ce qui ne rentre pas dans les categories ci-dessus |

#### 3. Date et heure

La date et l'heure actuelles sont pre-remplies. Modifiez si l'incident s'est produit plus tot.

#### 4. Description

Decrivez precisement ce qui s'est passe : circonstances, dommages constates, tiers impliques. **Minimum 20 caracteres.**

#### 5. Localisation (optionnel)

Saisissez le lieu manuellement (ex. : "A6 sortie 12") ou appuyez sur le bouton **GPS** pour recuperer automatiquement votre position.

#### 6. Photo (optionnel mais recommande)

Appuyez sur **"Prendre une photo"** pour utiliser l'appareil photo de votre telephone. Vous pouvez aussi choisir une image de votre galerie.

#### 7. Envoyer

Appuyez sur **"Signaler l'incident"**. Votre gestionnaire de flotte est immediatement notifie.

> **Assistance immediate** : si vous etes immobilise et avez besoin d'aide, cochez la case **"Besoin d'assistance immediate"**. Un bouton d'appel d'urgence apparait. L'incident sera marque comme priorite CRITIQUE.

## 6. Utiliser le module SOS en cas de panne

Le module SOS vous aide a trouver la bonne solution en cas de panne, en **4 questions**.

### Acceder au module SOS

1. Ouvrez FleetMaster depuis votre ecran d'accueil.
2. Allez dans le menu puis appuyez sur **SOS Garage**.

> Note : le module SOS est accessible aux utilisateurs connectes (gestionnaires et chauffeurs avec compte).

### Les 4 questions

#### Question 1 : Quel vehicule est en panne ?

Selectionnez votre vehicule dans la liste.

#### Question 2 : Quel type de probleme ?

| Option | Exemples |
|--------|----------|
| **Pneu** | Crevaison, pneu creve |
| **Mecanique** | Moteur, transmission |
| **Groupe frigo** | Groupe froid en panne |
| **Hayon** | Hayon elevateur bloque |
| **Accident** | Choc, carrosserie |

#### Question 3 : A quelle distance du depot ?

- **Moins de 50 km** : zone habituelle.
- **Plus de 50 km** : hors perimetre.

#### Question 4 : Le vehicule peut-il encore rouler ?

- **Roulant** : il peut rouler lentement.
- **Immobilise** : il ne bouge plus.

#### Localisation (optionnel)

Indiquez ou vous etes (ex. : "Aire d'autoroute A4, sortie 45").

### Le resultat

Appuyez sur **"Trouver une solution"**. Le systeme analyse votre situation et vous propose la meilleure option :

- **Contrat d'urgence** : si votre entreprise a un contrat avec un prestataire pour ce type de panne.
- **Assurance** : coordonnees et procedure de votre assurance.
- **Garage partenaire** : un garage du reseau FleetMaster.
- **Garage externe** : un garage a proximite si aucun partenaire n'est disponible.

Chaque resultat affiche les **coordonnees de contact** et les instructions a suivre.

> Si aucune solution automatique n'est trouvee, contactez votre assurance habituelle ou votre gestionnaire de flotte.

## 7. Consulter le carnet numerique du vehicule

Le carnet numerique regroupe tout l'historique d'un vehicule : inspections, pleins, maintenances, documents.

### Qui peut y acceder ?

L'acces au carnet est reserve aux **gestionnaires** (Admin, Directeur, Agent de parc). Si vous etes chauffeur, la carte "Carnet d'entretien" apparait verrouillee apres le scan QR.

> Contactez votre gestionnaire si vous avez besoin d'une information sur l'historique de votre vehicule.

### Pour les gestionnaires : comment consulter

1. Scannez le QR code du vehicule.
2. Appuyez sur la carte **"Carnet d'entretien"** (carte violette).
3. Si vous n'etes pas connecte, vous serez redirige vers la page de connexion, puis ramene automatiquement au carnet.

### Ce que contient le carnet

#### Vue d'ensemble

- **Kilometrage** actuel du vehicule.
- **Consommation moyenne** calculee sur les derniers pleins.
- **Cout carburant** total et volume total (en litres).
- **Etat des documents** : assurance et controle technique (valide ou expire).
- **Derniere inspection** : date, score et nom du conducteur.
- **Dernier plein** : date, quantite, prix et consommation.

#### Onglet Inspections

Historique de toutes les inspections avec pour chacune :
- La date.
- Le score (en pourcentage).
- Le nom du conducteur.
- Le statut (termine ou en cours).

#### Onglet Carburant

Historique de tous les pleins avec :
- La date.
- La quantite en litres.
- Le prix total.
- La consommation en L/100 km (si calculable).

#### Onglet Maintenance

Historique de toutes les interventions de maintenance :
- Le type d'intervention.
- La description.
- Le statut (terminee, en cours, planifiee).

#### Onglet QR Code

Affiche le QR code du vehicule que vous pouvez reimprimer ou partager.

## En cas de probleme

- **L'application ne s'installe pas** : verifiez que vous utilisez Chrome (Android) ou Safari (iPhone). Videz le cache du navigateur et reessayez.
- **Le QR code ne fonctionne pas** : verifiez que le code n'est pas abime ou masque. Si le probleme persiste, signalez-le a votre gestionnaire.
- **"Token invalide" apres un scan** : le QR code a peut-etre ete regenere. Demandez un nouveau QR code a votre gestionnaire.
- **Le kilometrage est refuse** : vous ne pouvez pas saisir un kilometrage inferieur au dernier enregistre. Verifiez le compteur du vehicule.
- **Pas de connexion internet** : les formulaires necessitent une connexion. Si vous etes dans une zone blanche, attendez d'avoir du reseau pour envoyer vos donnees.
